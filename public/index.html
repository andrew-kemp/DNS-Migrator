How to create a Cloudflare API Token
  1. Log in to dash.cloudflare.com/profile/api-tokens
  2. Click Create Token
  3. Find the Edit zone DNS template and click Use template
  4. Under Permissions, you should see Zone > DNS > Edit. Add a second permission:
    • Click + Add more
    • Select Account > Account Settings > Read
  5. Under Zone Resources, choose Include > All zones (or limit to specific zones)
  6. Click Continue to summary, then Create Token
  7. Copy the token and paste it above — you won't be able to see it again

Required permissions summary:

  • • Zone > DNS > Edit — create/update DNS records
  • • Zone > Zone > Edit — create new zones (included in template)
  • • Account > Account Settings > Read — list your accounts
